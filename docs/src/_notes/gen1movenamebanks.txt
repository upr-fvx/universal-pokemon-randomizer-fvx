
== Gen 1 move name bank offsets ==

We want to be able to repoint the move names. 

To do so, we must find where the games refer to the bank of the move names.

We start by grep-ing "MoveNames" in the pret/pokeyellow repository, finding four usages of BANK(MoveNames).
The Randomizer has tooling for finding offsets across all games, if we have a hex search string,
and a relative offset to the start of that hex search string.

Below, we have identified those two variables for the four locations.

NN: knowable byte, constant between games
XX: unknowable byte, varying between games
!!: target byte, whose offset we are looking for

search = FE 50 28 04 12 13 18
offset = -15

This search results in two different offsets, because .copyNameLoop is duplicated elsewhere.
The first offset in ROM / the one whose bank coincides with the other searches is the relevant one.
